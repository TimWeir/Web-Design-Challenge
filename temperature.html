    Temperature

    The purpose of this plot is to show the temperature across the various cities by latitude. Based on the Open Weather API and the CitiPy Python Libraries our set of 500 randomly selected cities from around the world was used create the various plots diplayed.

    This plot shows maximum temperature along the latitude lines for the cities in our data set. This result should be unsurprising as the closer to 0 latitude the more solar exposure an area would have.
